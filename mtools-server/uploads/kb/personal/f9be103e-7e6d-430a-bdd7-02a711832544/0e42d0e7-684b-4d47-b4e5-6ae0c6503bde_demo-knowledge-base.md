# 51ToolBox 使用指南

## 快速开始

51ToolBox 是一款桌面效率工具，集成了 AI 对话、插件系统、知识库检索等功能。

### 安装与启动

1. 从官方下载安装包
2. 双击安装后启动应用
3. 首次使用需要注册或登录账号

### 搜索栏

搜索栏是 51ToolBox 的核心交互入口：

- **普通搜索**：直接输入关键词，搜索已安装的插件和工具
- **AI 对话**：输入 `ai ` 前缀，快速进入 AI 对话模式
- **网页搜索**：输入 `gg ` 前缀，使用 Google 搜索
- **百度搜索**：输入 `bd ` 前缀，使用百度搜索
- **数据查询**：输入 `data ` 前缀，查询本地数据

## AI 功能

### 配置 AI 模型

在管理中心 -> AI 模型设置中，可以配置以下信息：

- **Base URL**：API 端点地址（支持 OpenAI 兼容接口）
- **API Key**：模型访问密钥（本地加密存储）
- **模型名称**：如 gpt-4o、claude-3.5-sonnet 等

### 团队 AI 配置

加入团队后，可以使用团队提供的 AI 配置，无需自己配置 API Key：

1. 团队管理员在管理中心配置 AI 模型
2. 团队成员自动获得访问权限
3. 支持按成员设置 token 用量限制

### 知识库检索增强（RAG）

将文档导入知识库后，AI 对话时可以自动检索相关内容，提供更精准的回答：

- 支持格式：txt、md、json、csv、html
- 自动分块和向量化处理
- 可调节检索参数（topK、相似度阈值等）

## 插件系统

### 内置插件

| 插件 | 功能 |
|------|------|
| 屏幕取色 | 获取屏幕任意位置颜色值 |
| 截图工具 | 区域截图、OCR 文字识别 |
| 知识库 | 文档管理与 RAG 检索 |
| 管理中心 | 账号、团队、AI 配置管理 |
| 云同步 | 多设备数据同步 |

### 开发自定义插件

51ToolBox 支持通过插件接口扩展功能：

```typescript
interface MToolsPlugin {
  id: string;
  name: string;
  description: string;
  category: string;
  render: (props: MToolsPluginProps) => ReactNode;
}
```

## 团队协作

### 创建团队

1. 进入管理中心 -> 我的团队
2. 点击"创建团队"
3. 邀请成员（通过邮箱或手机号）

### 团队角色

- **Owner**：团队创建者，拥有全部权限
- **Admin**：管理员，可管理成员和配置
- **Member**：普通成员，可使用团队资源

### 资源共享

团队成员可以共享以下资源：
- 知识库文档
- 工作流配置
- AI 模型配置

## 数据同步

支持多种同步方式：

- **mTools Cloud**：官方云同步服务
- **GitHub/Gitee/GitLab**：通过 Git 仓库同步
- **WebDAV**：兼容 WebDAV 协议的网盘（如坚果云）

同步内容包括：书签、代码片段、工作流、笔记、标签等。

## 常见问题

### Q: API Key 安全吗？
A: 所有 API Key 均使用 AES-256-GCM 加密存储，密钥由设备唯一标识派生，不会明文暴露。

### Q: 知识库支持哪些语言？
A: 支持中英文混合内容，分块算法对中文做了专门优化。

### Q: 能否离线使用？
A: 搜索、插件等基础功能可离线使用。AI 对话和云同步需要网络连接。
